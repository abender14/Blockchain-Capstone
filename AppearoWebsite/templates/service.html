Our Creditworthiness Model

Machine Learning is a data analysis tool that enables a computer to make predictions based on learned data and experiences.



Benefit

We use machine learning algorithms to predict - at scale - the probability of a borrower defaulting on a loan. This technology is essential to providing all of our customers a fast, safe, and accurate creditworthiness assessment.

Dataset

Appearo’s model is based on several supervised machine learning algorithms that predict probabilities based on inherent patterns in the data. The dataset used to train our model is maintained by Kaggle, and contains the loan records for 150,000 borrowers.

Limitations

The currently displayed model (below) only accounts for a borrower’s financial data from the training dataset. The commercialized model will use alternative factors as inputs for a new training dataset.
